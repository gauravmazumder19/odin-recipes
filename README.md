# Odin Recipes

This is my **first basic HTML project** made while learning web development from **The Odin Project**.

It is a simple website that shows a list of recipes and links to their individual pages.

## Live Demo

https://gauravmazumder19.github.io/odin-recipes/
---

## What I Learned

- Basic HTML structure  
- How to use headings and lists  
- How to create links between pages  
- Folder structure for websites  

---

## Recipes Included

- Butter Chicken  
- Mango Lassi  
- Kaju Katli  

Each recipe has its own page with ingredients and steps.

---

## Built With

- HTML5  
- No CSS or JavaScript (pure beginner project)

---

## Purpose

This project was created to practice:
- Writing clean HTML
- Understanding how webpages connect using links
- Getting comfortable with web project structure
